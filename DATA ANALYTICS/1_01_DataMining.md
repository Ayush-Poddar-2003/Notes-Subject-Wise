

**Data ?**   
Raw facts and figures.  
No context or meaning.


**Information ?**  
Processed and meaningful data.  
Adds context for decision-making.



## What Is Data Mining?

Data mining is the process of discovering  
**patterns, trends, correlations, or useful knowledge**  
from large datasets  
using statistical, ML, and DB techniques.

> Extracting hidden knowledge from huge data.

Examples
- Predicting customer behavior  
- Fraud detection  
- YouTube recommendations  
- Market analysis  

---

### Kinds of Data That Can Be Mined -

**1. Database Data ?**  
Structured data stored in tables (rows/columns).  

**2. Data Warehouses ?**  
Large, historical, integrated datasets, 
Stored in multidimensional structures (data cubes).  

**3. Transactional Data ?**  
Generated from real-world transactions.  
Contains : Transaction ID,  Timestamp,  Items purchased,  Amount  

**4. Other Kinds of Data ?**  
**Spatial Data**: Location-based data, maps, GPS.  
**Temporal Data**: Time-series: stock prices, sensor readings.  
**Text Data**: Emails, documents, chats.  
**Multimedia Data**: Images, audio, video.  
**Web Data**: Website logs, clickstreams, browsing history.  
**Biological Data**: DNA sequences, genomic data.
