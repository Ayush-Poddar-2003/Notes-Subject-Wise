# Mobile Agents
A software program that can:
- Move from one device to another
- Perform tasks automatically
- Gather information
- Communicate with services
- Continue working even when the user is offline

---
Services Offered by Mobile Agents
- Information collection
- Monitoring tasks
- Network testing
- Price comparison
- File transfer
- Searching large networks

---
Features
1. Mobility → moves across networks
2. Autonomy → works without direct control
3. Communication → interacts with other agents
4. Learning → can adapt
5. Cooperation → agents work in teams
