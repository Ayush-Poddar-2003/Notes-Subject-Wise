# NETWORK GENERATIONS

| Generation | Technology      | Features                                   |
| ---------- | --------------- | ------------------------------------------ |
| **1G**     | Analog Cellular | Voice only                                 |
| **2G**     | GSM, CDMA       | Digital voice, SMS                         |
| **2.5G**   | GPRS, EDGE      | Packet data, basic internet                |
| **3G**     | UMTS, CDMA2000  | High-speed data, multimedia                |
| **4G**     | LTE             | Broadband internet, video streaming        |
| **5G**     | NR (New Radio)  | Ultra-low latency, massive IoT, high-speed |


---
# GSM ARCHITECTURE
Global System for Mobile Communications (GSM) is a 2G digital standard.

Components:
- Mobile Station (MS)
- Base Station Subsystem (BSS)
- Network Subsystem (NSS)
- Operation Support System (OSS)


---
# GPRS (2.5G)
General Packet Radio Service  
Provides packet-switched data for GSM.  
Enables internet browsing, email, and MMS.

---
# 3G MOBILE SERVICES
Based on UMTS (Universal Mobile Telecommunication System)  
Supports voice, video calls, and high-speed mobile internet.  
Part of IMT-2000 standard (International Mobile Telecommunications).


---
# 4G LTE
Long-Term Evolution (LTE) is the 4G standard.
Features:
- IP-based network
- High-speed internet
- HD video streaming
- Low latency

---
# 5G Mobile Networks
Fifth-generation networks with:
- Ultra-reliable, low-latency communication
- Massive IoT connectivity
- Enhanced mobile broadband
- Network slicing for multiple services
