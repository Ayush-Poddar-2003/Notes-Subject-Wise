## Using Protégé (Ontology Editor)

Protégé is the official free tool from Stanford.

---
#### What you do in Protégé ?  
Create classes  
Add properties  
Define restrictions  
Add individuals  
Run reasoners (HermiT, Pellet)  
Write SPARQL queries  
Export ontology in OWL/RDF/XML/TTL

---
### Important Protégé Tabs:  
Entities → Classes  
Object properties  
Data properties  
Individuals  
Reasoner  
SPARQL

---
### Why Protégé is popular?  
Easy GUI  
Supports OWL  
Visualization plugins
